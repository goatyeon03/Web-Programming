## 여행지 정보 제공 사이트

사용자가 원하는 지역을 입력하면
주변 여행지 / 카페 / 음식점 정보를 지도 기반으로 제공하는 웹 서비스입니다.
Flask 기반의 백엔드와 HTML 템플릿을 활용하여 간단한 웹 애플리케이션을 구현했습니다.

### 1. 주요 기능
- 📍지역 기반 장소 검색<br>사용자가 검색창에 입력한 지역을 기준으로 주변 장소 정보 제공
- 🗺지도 시각화<br>지도 상에 위치 정보를 마커로 표시
- 🏠카테고리별 장소 확인<br>카페, 음식점 등 카테고리별로 정보 확인 가능
- 🌐웹 인터페이스 제공<br>바로 해당 링크에 접근 가능한 형태


### 2. 프로젝트 구조

### 3. 사용방법
1. 메인화면에서 '카페', '식당', '관광지' 중 검색을 원하는 것을 클릭합니다.
2. 이동된 페이지에서 검색창에 원하는 동네를 입력합니다.
3. 구글맵 상에 해당 동네에 있는 인근 '카페' or '식당' or '관광지'가 마크 표시가 됩니다.
4. 특정 마크를 클릭하면 장소 정보가 뜹니다. (위치, 리뷰, 영업시간 등 - places api에서 제공하는 것)
5. 지도 옆에 해당 동네(혹은 특정 카페, 식당, 관광지)에 대한 네이버 블로그 게시물들을 찾아 카드 형태로 제공합니다.
6. 필터 버튼을 통해 원하는 단어를 포함하는 게시물만 남길 수 있습니다.
7. 카드를 클릭하면, 카드 뒷면이 나타나고 여기에는 게시물 요약문과 링크가 있습니다.
8. 링크를 클릭하면 해당 블로그로 이동할 수 있습니다.
